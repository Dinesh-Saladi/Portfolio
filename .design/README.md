# Portfolio Design System — Reference Guide

> **Inspiration**: olivierlarose.com  
> **Style**: Dark editorial, typography-driven, animation-rich, minimal  
> **Built with**: Next.js (App Router) + TypeScript + Tailwind CSS v4 + Framer Motion + Lenis + GSAP

---

## Quick Reference

| Aspect | Value |
|--------|-------|
| Background | `#0a0a0a` |
| Foreground | `#e8e8e8` |
| Accent | `#999` |
| Muted | `#555` |
| Primary Font (Display) | Cormorant Garamond (serif) |
| Primary Font (Body) | Inter (sans-serif) |
| Fallback Font | Geist (local) |
| Global Easing | `[0.76, 0, 0.24, 1]` cubic-bezier |
| Smooth Scroll | Lenis (duration: 1.2) |
| Animation Library | Framer Motion |
| Layout Approach | 12-column grid, px-6 md:px-10 |
| Overall Margin | m-1 (4px) around main content |
| Mode | Dark only |
